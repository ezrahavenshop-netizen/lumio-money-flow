Make both the admin and user dashboard sidebar 
fully responsive with a collapse toggle.

===================================================
USER DASHBOARD SIDEBAR
===================================================

Add a collapse toggle button at the bottom of
the sidebar above the avatar/logout section:

Button:
  Icon: ChevronLeft (Lucide) when expanded
  Icon: ChevronRight (Lucide) when collapsed
  Small square button, gold border, navy background
  Hover: gold background, white icon

Expanded state (default, 240px wide):
  Shows icon + full label for each nav item
  Shows full avatar + name + "Premier Account"
  Shows full Lumio logo

Collapsed state (64px wide):
  Shows icons only, no labels
  Shows avatar circle only, no name
  Shows Lumio icon/symbol only, no wordmark
  Each icon centered in the 64px width
  Hover on icon: gold tooltip appears on right
  showing the nav item name

Transition:
  Smooth width animation 300ms ease
  Labels fade out on collapse
  Labels fade in on expand

State persists during the session:
  If user collapses sidebar and navigates
  to another page sidebar stays collapsed

Main content area:
  Expands to fill remaining width when
  sidebar collapses
  Contracts when sidebar expands
  Smooth transition 300ms ease
